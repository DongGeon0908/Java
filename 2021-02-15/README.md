### 📒Contents

- 메소드
    + 블록에 있는 모든 코드들이 일괄적으로 실행
    + 필드를 읽고 수정하는 역할
    + 다른 객체를 생성해서 다양한 기능을 수행
    + 객체 간의 데이터 전달의 수단
    + 외부로부터 매개값을 받을 수 있음
        * 실해 후 어떤 값을 리턴할 수 있음
    
- 메소드 선언
    + 선언부 `signature`
        * 리턴타입
        * 메소드이름
        * 매개변수선언
    + 실행 블록

- 리턴 타입
    + 메소드가 실행 후 리턴하는 값의 타입

- 매개변수를 `...`로 선언
    + 메소드 호출 시 넘겨준 값의 수에 따라 자동으로 배열이 생성되고 매개값으로 사용
